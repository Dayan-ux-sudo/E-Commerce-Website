###This is a E-Commerce Web Application ###
This is a basic e-commerce web application where one can trade and shop online.


##Features##
It has a shoping cart used to add/remove items and also used for quantity mangement.  
It has a responsive design.
There is databse integration using MySQL with PDO.
It also has user authentication where a user can register and login to the web application.


##To Get Started:##
Create the database using the provided SQL
Update database credentials in config.php
Add some sample products to the database
Upload product images to assets/images/ folder
Run the website on a local server (XAMPP, WAMP, or MAMP)
